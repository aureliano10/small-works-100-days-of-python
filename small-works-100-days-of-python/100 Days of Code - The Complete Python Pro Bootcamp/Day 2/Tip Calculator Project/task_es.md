Vamos a construir una calculadora de propinas.

Si la cuenta fue de $150.00, dividida entre 5 personas, con un 12% de propina. 

Cada persona debería pagar:

`(150.00 / 5) * 1.12 = 33.6`

Después de formatear el resultado a 2 decimales = `33.60`
